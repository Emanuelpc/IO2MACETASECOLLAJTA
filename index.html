¿Cómo funciona?

Puedes ingresar CUALQUIER dato que conozcas (solo uno es suficiente).

Ejemplos:

  • Si solo sabes el tiempo disponible (ej: 3 horas), calcularé cuántas macetas puedes hacer
  • Si solo sabes la cantidad de cáscara, calcularé cuántas macetas puedes producir
  • Si solo sabes el personal disponible, calcularé la producción posible
  • ¡Y así con cualquier dato!

Usa 0 para los datos que no conozcas. El sistema calculará automáticamente los valores faltantes.
